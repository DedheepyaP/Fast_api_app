FastAPI CRUD Application (Dockerized)

This project is a containerized CRUD application built using FastAPI, PostgreSQL, and Streamlit UI, orchestrated with Docker Compose.

Tech Stack:

  Backend: FastAPI, SQLAlchemy
  Database: PostgreSQL
  Frontend: Streamlit
  Containerization: Docker & Docker Compose
  

--Prerequisites--
  Docker
  Docker Compose

--Run the Application--
Build and start the containers:
>> docker compose up --build

--Access the application--
FastAPI API: http://localhost:8000
Streamlit UI: http://localhost:8501
